# 缩略图

## Overview

定义缩略图来源、选择策略、规格与一致性校验标准。

## 操作指南 / 工作流程

## 1. 获取与选择

- 来源：Web Search（必须使用 Web Research Server MCP）
- 策略：风格一致性与质量优选
  - 必须确保搜索到合适图片，结果不佳需优化搜索词重试
  - 每个模型搜索 3–5 张候选图
  - 优先风格：3D 渲染图 > 干净背景实拍图 > 实拍图
  - 背景：白色/灰色/透明
  - 视角：三分之四前视角的轻俯视优先
  - 优选标准：分辨率 > 800px，主体完整无遮挡
  - 自动化：下载候选图，按分辨率与文件体积择优

## 2. 去重要求

- 对比范围：`models/*/{ModelName}/{ModelName}.png|jpg` 与 `models/*/{ModelName}/{ModelName}_mil.png`
- 判定建议：pHash 汉明距离 < 8 或 SSIM ≥ 0.75 视为高相似

## 3. 规格与命名

- 格式：PNG/JPG
- 命名：`{ModelName}.png`
- 禁止项：严禁使用 `*_mil.png` 作为缩略图
- 兜底顺序：高质量公开版权实拍/渲染图 > GLB 离线渲染 > 暂停打包并报错

## 4. 一致性校验 (Thumbnail vs 3D)

- 从 GLB 离线渲染预览图（等轴侧视角，白/灰背景，≥1024px）
- 相似度指标：
  - SSIM ≥ 0.45
  - 边缘轮廓 IoU ≥ 0.60
  - 颜色直方图相关性 ≥ 0.50
- 判定准则：三项指标至少满足两项
- 自动修复：相似度不足则用离线渲染图替换缩略图并更新 `agent.json.thumbnail`
- 禁止打包：无法生成离线渲染图或相似度仍严重不足
