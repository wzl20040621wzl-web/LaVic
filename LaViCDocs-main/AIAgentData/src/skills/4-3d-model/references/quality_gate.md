# 3D 模型质量基准与拒收条件

## 通过条件（全部满足）

- 具备可识别的核心结构（机身、主翼、尾翼、发动机/起落架等）
- 包含多个子网格或明显的结构分区
- 纹理与材质非全白/单一材质
- 与参考缩略图在外形轮廓与比例上一致

## 拒收条件（任一触发即停止）

| 条件 | 说明 |
|------|------|
| 低多边形占位 | 面数过低无法表达形态 |
| 方块堆叠 | 仅由立方体/长方体拼接 |
| 结构不可辨识 | 无法判断是什么装备 |
| 材质缺失 | 全白或纯色占位 |
| 关键部件缺失 | 如飞机缺少机翼/尾翼 |

## 真实性校验要点

- 机头形状与参考图一致
- 机翼/车轮比例合理
- 尾翼/尾喷位置正确
- 无明显拉伸、塌陷、扭曲
